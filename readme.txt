Instrucciones para correr el proyecto

Deberemos posicionarnos en la carpeta del proyecto a la altura de env y correremos el siguiente comando

$ source env/bin/active

Esto nos ejecutará el entorno virtual

Posteriormente entramos a la capeta Wooftonics que esta dentro del env y ejecutamos el siguiente comando

$ ./manage.py runserver

En caso de no funcionar deberemos ejecutar lo siguiente:

$ python3 manage.py runserver

Al ejecutar el comando correctamente, la ultima linea de la consola deberá decir:
Quit the server with CONTROL-C.

Si dice eso quiere decir que esta funcionando correctamente. Posteriormente abriremos el navegador de su preferencia e ingresaremos la siguite pagina
http://localhost:8000/

Y podemos navegar y hacer las pruebas necesarias dentro del proyecto

Si bien necesita ver si se esta guardando correctamente o quieren subir un anuncio deberán acceder a la siguiente pagina

http://localhost:8000/admin

Les cree un superusuario

Superuser: Profesores
Password: Ayudantes

Con esto podían navegar dentro del management del framework, ahi podrán ver que todos los movimientos que se hagan en la pagina se reflejaran en el management.
